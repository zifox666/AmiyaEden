# 自动审批机器人

## 基础功能

1. 修改舰队 - 舰队配置 - 装配 - 可以查看装配 -> 非必要装备/可替换装备/必要装备(默认必要 | 装备不符选项（补损减半/不补损）)
2. 修改创建舰队 -> 新增是否启用自动审批（不启用/仅提交/自动审批）
3. 舰队pap后 - 根据当前舰队选择的舰队类型 - 启用自动流程
4. 仅提交：自动提交当前时间段当前舰队角色的KM到SRP系统(需要ship_type_id在舰队配置中，且考虑pap后killmail刷新的问题)
5. 自动审批：自动提交当前时间段当前舰队角色的KM到SRP系统，并且如果符合条件则自动审批通过(需要ship_type_id在舰队配置中，且考虑pap后killmail刷新的问题)

## 可替换装备

1. 在舰队配置处新增可替换装备的功能，用户可以在舰队配置中指定某个必要装备的可替换装备列表。当用户提交KM时，如果KM中的装备不符合必要装备的要求，但符合可替换装备的要求，则仍然可以通过审批

## 装备检测

1. 当前存储的装备是HISlot123 但是理论上装备在能量槽具体槽位不同可以以被接受 而不是必须 HiSlot1 == HISlot1 但是 HiSlot1 可以是 HISlot2 HISlot3 HISlot4 HISlot5 HISlot6 只要其中一个满足就可以通过审批
2. 装备可以多 但是不能少 比如HiSlot理论上有五个这个装备 但是KM里只有四个 那么这个装备就不满足要求 但是如果KM里有六个 那么这个装备就是满足要求的
3. 在补损审批备注需要加上所有不符的装备信息
4. 仓库/无人机不加入检测

## 补损价格

1. 当前补损价格表是一个笼统的 舰队配置里的是指定了当前舰队的价格 如果舰队配置的是0 那么就用全局的价格表 如果舰队配置里有价格 那么就用舰队配置里的价格
