# EVE 补损管理系统

## 概述

用户可以通过本模块选择系统refresh模块存储的KM，并选择参与的舰队/备注后提交补损申请，管理员(srp/当前舰队fc/admin)可以在后台审批通过后发放PAP奖励。

## 功能列表

1. 提交补损

    1. 通过快捷链接：用户使用带fleet_id的快捷链接访问补损申请页面，系统会自动识别fleet_id并预填相关信息(自动筛选符合舰队时间和参与过的角色段内的KM)。
    2. 手动选择：用户可以在补损申请页面手动选择参与的舰队或者填写备注信息（没有舰队就填写备注/有就是舰队）。

2. 审批流程

    1. 审批/发放状态 分开
    2. 须显示角色/舰船/星系/时间/角色军团联盟/killid/推荐补损金额(可单独修改)/同时支持KM预览
    3. 补损金额表：创建一张(srp/admin)可编辑的表格来对应ship_id/标准补损金额
